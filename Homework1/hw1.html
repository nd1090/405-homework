      • alink: Color of text for hyperlinks when selected.

  4. Unordered List Element
    • Official Unordered List Element Document
    • Attributes
      • compact: This Boolean attribute hints that the list should be rendered in a compact style. The interpretation of this attribute depends on the user agent and it doesn't work in all browsers.
      • Type: Used to set the bullet style for the list.
  5. Anchor Element
    • Official Anchor Element Document
    • Attributes
      • download: This attribute instructs browsers to download a URL instead of navigating to it, so the user will be prompted to save it as a local file. If the attribute has a value, it is used as the pre-filled file name in the Save prompt (the user can still change the file name if they want). There are no restrictions on allowed values, though / and \ are converted to underscores. Most file systems limit some punctuation in file names, and browsers will adjust the suggested name accordingly.
      • href: Contains a URL or a URL fragment that the hyperlink points to.
      • ping: Contains a space-separated list of URLs to which, when the hyperlink is followed, post requests with the body ping will be sent by the browser (in the background). Typically used for tracking
  6. Image Embed Element
    • Official Image Embed Element Document
    • Attributes
      • alt: This attribute defines an alternative text description of the image.
      • Crossorigin: This enumerated attribute indicates if the fetching of the related image must be done using CORS or not.
      • Decoding: Provides an image decoding hint to the browser.

Vocab Definitions

  1. HTTP
    • Definition: Hypertext Transfer Protocol (HTTP) is how computers interpret the information we give them and how they communicate with each other through the web.
    • Source: Official HTTP Document
  2. HTML
    • Definition: Hypertext Markup Language (html) is the specific Language that makes up the base structure of a webpage.
    • Source: Official HTML Document
  3. Element
    • Definition: An element is a part of a webpage. It consists of openeing and closing tags with content inbetween.
    • Source: Official Element Document
  4. Tag
    • Definition: A tag is used for making an element. These are placed in angle prackets and always have a closing tag (which is the same as the opening tag except it is preceded by a "/").
    • Source: Official Tag Document
  5. Block (CSS)
    • Definition: A block in HTML is an element that appears on a new line. EX: The paragraph element is a block element, but the anchor element is an inline element.
    • Source: Official Block (CSS) Document
  6. Doctype
    • Definition: "Doctype" appears at the top of every html page to ensure that the browser reads the code correctly.
    • Source: Official Doctype Document
  7. Head
    • Definition: This contains metadata such as an author or a link to CSS files that are to be applied to HTML.
    • Source: Official Head Document
  8. Attribute
    • Definition: An attribute extends a tag and it changes the tags behavior or provides metadata.
    • Source: Official Tag Document
  9. Boolean
    • Defintion: a computer logical data type that only consists of two values, true and false.
    • Source: Official Boolean Document
  10. DOM
    • Definition: The DOM (Document Object Model) is an API (Application Programing interface) that interacts with HTML documents.
    • Source: Official DOM Document
  11. CSS
    • Definition: CSS (Cascading Style Sheets) controls how information looks on a webpage.
    • Source: Official CSS Document
  12. Property (CSS)
    • Definition: Property (CSS) is how a browser displays a certain value (EX: adding color to text)
    • Source: Official Property (CSS) Document
  13. Selector (CSS)
    • Definition: CSS selector is a CSS rule that describes what properties elements in a document will posess (EX: font and size).
    • Source: Official Selector (CSS) Document
